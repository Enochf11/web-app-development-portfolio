Business Showcase

This page highlights example projects that demonstrate how I use digital tools to respond to real business problems. Each project connects web development, data, and decision-making.

1. Restaurant Ordering Concept

A web-based ordering interface for a casual dining restaurant. The project focuses on transparent pricing, mobile-friendly design and a clear user journey from choosing menu items to seeing an estimated total.

From a business point of view, this helps reduce confusion about prices, supports better communication with customers and can increase trust. It also creates a foundation for collecting data on popular menu items in future.

Key skills: HTML, CSS, JavaScript, UX, basic pricing logic.

2. Business Intelligence Dashboard Concept

A concept for a manager dashboard that shows key performance indicators such as daily sales, table occupancy, average spend per customer and customer feedback.

The aim is to turn raw data into clear visual insights that support decision-making. For example, a manager could quickly see which days are busiest, which dishes sell the most, or whether certain promotions are working.

In a real implementation, a tool like Power BI or similar would be used to connect data sources and create interactive charts.

Key skills: Data analysis, KPIs, dashboard design, decision support.

3. Personal Portfolio Website

This website itself is a business-focused project. It is designed to position me as a student developing skills in web app development and digital business. The structure, branding and content are planned to be clear and professional so that an employer or client can quickly understand who I am and what I can do.

The navigation links to my skills, contacts, certificates and problem-solving case study. This shows how a simple web presence can support personal branding and professional opportunities.

Key skills: Branding, communication, front-end design, reflection.
